c.reco v1.0.0

REQUIREMENTS
------------
1. JDK 7 or later
2. Apache Tomcat 8 or later

INSTRUCTIONS
------------
1. Extract the ZIP archive to any location.
2. Ensure that Tomcat server is not running.
3. Copy the WAR archive to "webapps" directory of Tomcat.
4. Start Tomcat server.
5. Go to localhost:8080/creco
6. Enjoy!

NOTES
-----
1. Crawled pages are stored in the "content" directory in web app.
2. Index files are stored in the "index" directory in web app.
